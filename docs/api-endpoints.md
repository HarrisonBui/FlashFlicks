# API Endpoints

## HTML API

### Root

- `GET /`

## JSON API

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Movies

- `GET /api/movies`
- `GET /api/movies/:movie_id`

### MovieLists

- `GET /api/movielists`
- `POST /api/movielists`
- `DELETE /api/movielists/:id`

### MovieListMovies

- `POST /api/movie_list_movies`

### WatchStatus

- `GET /api/watch_status`

### WatchStatusMovies

- `POST /api/watch_status_movies`

### Reviews

- `GET /api/reviews/`
- `POST /api/reviews/`
